═══════════════════════════════════════════════════════════════
   РУЧНАЯ НАСТРОЙКА
═══════════════════════════════════════════════════════════════

1. File → Settings (Ctrl+Alt+S)

2. Project: word_generator → Python Interpreter

3. Если интерпретатор не выбран:
   - Нажми на шестерёнку ⚙️ (Settings)
   - Выбери "Add..."

4. В окне "Add Python Interpreter":
   - Выбери "Existing environment"
   - Нажми на "..." рядом с "Interpreter"
   - Укажи путь:
     C:\Users\Hammer_1983\PycharmProjects\word_generator\generator\venv\Scripts\python.exe
   - Или просто выбери папку venv\Scripts\python.exe
   - Нажми OK

5. Нажми OK → Apply → OK
