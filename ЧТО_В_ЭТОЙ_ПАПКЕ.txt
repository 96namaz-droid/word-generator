    ЧТО НАХОДИТСЯ В ЭТОЙ ПАПКЕ
═══════════════════════════════════════════════════════════════════

Эта папка содержит ВСЁ необходимое для запуска программы
"Генератор Word-отчётов о испытании пожарных лестниц"
на любом компьютере с Windows.

═══════════════════════════════════════════════════════════════════

📂 СТРУКТУРА ПАПКИ:
───────────────────

📁 generator/
   │
   ├─ 📄 START.bat ⭐ ЗАПУСК ПРОГРАММЫ
   ├─ 📄 INSTALL.bat ⭐ УСТАНОВКА БИБЛИОТЕК
   │
   ├─ 📄 QUICK_START.txt ← НАЧНИТЕ С ЭТОГО ФАЙЛА
   ├─ 📄 INSTRUCTION.txt ← Полная инструкция
   ├─ 📄 README.txt ← Краткое описание
   │
   ├─ 📄 main.py ← Главный файл программы
   ├─ 📄 gui.py ← Интерфейс
   ├─ 📄 document_generator.py ← Генератор документов
   ├─ 📄 config.py ← Настройки (реквизиты компании)
   ├─ 📄 validator.py ← Проверка данных
   ├─ 📄 history_manager.py ← История
   ├─ 📄 logger.py ← Логирование
   ├─ 📄 requirements.txt ← Список библиотек
   │
   ├─ 📁 reports/ ← Сюда сохраняются готовые отчёты
   ├─ 📁 contracts/ ← Сюда можно положить договоры
   └─ 📁 logs/ ← Сюда записываются логи работы

═══════════════════════════════════════════════════════════════════

🚀 БЫСТРЫЙ СТАРТ:
─────────────────

1. Откройте файл QUICK_START.txt
2. Следуйте 3 простым шагам
3. Готово!

═══════════════════════════════════════════════════════════════════

⚙️ ЧТО ДЕЛАЕТ ПРОГРАММА:
─────────────────────────

✓ Автоматически создаёт Word-документы (протоколы испытаний)
✓ Рассчитывает нагрузку на балки по формуле
✓ Проверяет соответствие ГОСТ Р 54253-2009
✓ Автоматически формирует выводы
✓ Сохраняет историю заказчиков
✓ Проверяет правильность введённых данных

═══════════════════════════════════════════════════════════════════

📋 СИСТЕМНЫЕ ТРЕБОВАНИЯ:
────────────────────────

• Windows 7/8/10/11
• Python 3.8 или выше (нужно установить отдельно)
• Интернет (только для установки библиотек)
• 100 МБ свободного места

═══════════════════════════════════════════════════════════════════

🔧 ЧТО НУЖНО УСТАНОВИТЬ:
────────────────────────

1. Python (скачать с https://www.python.org/downloads/)
   ⚠ При установке отметить "Add Python to PATH"

2. Библиотеки Python (запустить INSTALL.bat)

Всё! Больше ничего не требуется.
