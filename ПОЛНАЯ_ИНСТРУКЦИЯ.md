# Полная инструкция по созданию репозитория и загрузке кода

## Вариант 1: Автоматический (рекомендуется)

### Шаг 1: Получите GitHub Personal Access Token

1. Откройте https://github.com/settings/tokens
2. Нажмите **"Generate new token (classic)"**
3. Введите название токена (например: "Word Generator Upload")
4. Выберите срок действия (рекомендуется: 90 дней)
5. Отметьте права: **"repo"** (полный доступ к репозиториям)
6. Нажмите **"Generate token"**
7. **ВАЖНО:** Скопируйте токен сразу! Он показывается только один раз.

### Шаг 2: Запустите автоматический скрипт

```bash
CREATE_AND_UPLOAD.bat
```

Скрипт попросит:
- Ваш GitHub username
- Название репозитория (или нажмите Enter для "word-generator")
- Ваш Personal Access Token
- Создать приватный репозиторий? (y/n)

**Готово!** Репозиторий будет создан и код загружен автоматически.

---

## Вариант 2: Через PowerShell напрямую

Если у вас уже есть токен, выполните:

```powershell
.\create_github_repo.ps1 -RepoName "word-generator" -GitHubUsername "ВАШ_USERNAME" -Token "ВАШ_ТОКЕН"
```

Для приватного репозитория добавьте `-Private`:

```powershell
.\create_github_repo.ps1 -RepoName "word-generator" -GitHubUsername "ВАШ_USERNAME" -Token "ВАШ_ТОКЕН" -Private
```

## Вариант 3: Вручную (без токена)

### Шаг 1: Создайте репозиторий на GitHub

1. Откройте https://github.com/new
2. Введите название: `word-generator` (или любое другое)
3. Выберите Public или Private
4. **НЕ добавляйте** README, .gitignore или лицензию
5. Нажмите **"Create repository"**

### Шаг 2: Загрузите код

Запустите:

```bash
AUTO_UPLOAD.bat
```

Или вручную:

```bash
git remote add origin https://github.com/ВАШ_USERNAME/word-generator.git
git branch -M main
git push -u origin main
```

## Безопасность токена

⚠️ **ВАЖНО:** Никогда не публикуйте ваш Personal Access Token в открытом доступе!

После использования токен можно отозвать на странице:
https://github.com/settings/tokens
