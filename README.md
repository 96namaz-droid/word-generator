# Генератор Word-отчётов

Приложение для автоматической генерации протоколов испытаний вертикальных пожарных лестниц, маршевых лестниц и ограждений кровли.

## Описание

Генератор Word-отчётов - это Python-приложение с графическим интерфейсом (Tkinter), которое позволяет автоматически создавать протоколы испытаний согласно ГОСТ Р 53254-2009 и ГОСТ Р 54253-2009.

## Возможности

- Генерация протоколов для вертикальных пожарных лестниц (тип П1-1, П1-2)
- Генерация протоколов для маршевых лестниц (тип П2)
- Генерация протоколов для ограждений кровли
- Автоматическое извлечение данных из договоров
- Автоматическое получение погодных данных
- Управление историей созданных отчётов
- Валидация вводимых данных

## Установка

1. Установите Python 3.8 или выше
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

### Графический интерфейс
```bash
python main.py
```

или

```bash
START.bat
```

### Веб-версия
```bash
python start_web.py
```

или

```bash
start_web.bat
```

## Структура проекта

- `main.py` - точка входа приложения
- `gui.py` - графический интерфейс
- `document_generator.py` - фасад для генерации документов
- `generators/` - генераторы протоколов для разных типов конструкций
- `config.py` - конфигурация приложения
- `contract_parser.py` - парсер договоров
- `validator.py` - валидация данных
- `history_manager.py` - управление историей
